# React Stack Example

Example stack for React.js written in Es2015. Bundlered by Webpack for develop with HotReload.

For tests is configured Karma runner using Mocha framework.


### Install Project

`npm install`

### Build Project
`npm run build`

###Develop Project
 `npm start`

 ### Single Run Tests
 `npm test`

 ### Develop Tests
 `npm run test:dev`
